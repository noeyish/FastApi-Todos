# 사용자 시나리오 & 사용자 여정 맵
## My TodoList 앱 — v5.0.0

---

## 1. 주요 사용자 시나리오

### 시나리오 1 — 신규 사용자 회원 가입 및 첫 할 일 등록

**행위자:** 김지은 (대학생, 신규 방문)
**전제 조건:** 계정 없음, 앱 첫 방문

| 단계 | 사용자 행동 | 시스템 반응 |
|------|------------|------------|
| 1 | 앱 URL 접속 | 로그인 화면 표시 |
| 2 | "회원가입" 탭 클릭 | 회원가입 폼 표시 |
| 3 | 이메일, 비밀번호 입력 후 가입 | 계정 생성, 자동 로그인 후 메인 페이지 이동 |
| 4 | 제목, 완료일, 우선순위 입력 후 저장 | 목록에 새 항목 즉시 표시 |
| 5 | 통계 대시보드 확인 | 전체 1개, 완료 0개, 완료율 0% 표시 |

**성공 기준:** 가입 ~ 첫 Todo 등록까지 5분 이내

---

### 시나리오 2 — 기존 사용자 일상적 할 일 관리

**행위자:** 박준호 (고등학생, 기존 사용자)
**전제 조건:** 계정 있음, 기존 Todo 항목 5개 존재

| 단계 | 사용자 행동 | 시스템 반응 |
|------|------------|------------|
| 1 | 앱 접속 후 로그인 | 본인의 Todo 목록 + 통계 표시 |
| 2 | 기한 초과 항목 확인 | 빨간 강조 항목 시각적으로 인식 |
| 3 | 완료한 항목 체크박스 클릭 | 취소선 처리, 통계 완료율 즉시 업데이트 |
| 4 | 제목 변경이 필요한 항목 인라인 수정 | 카드에서 바로 편집 후 저장 |
| 5 | 필요 없는 항목 삭제 | 확인 다이얼로그 → 삭제 후 목록 제거 |

**성공 기준:** 목록 관리 작업 전체 3분 이내 완료

---

### 시나리오 3 — 검색으로 빠른 항목 탐색 _(v5 신규)_

**행위자:** 김지은
**전제 조건:** Todo 항목이 10개 이상 존재

| 단계 | 사용자 행동 | 시스템 반응 |
|------|------------|------------|
| 1 | 검색창에 키워드 입력 | 입력과 동시에 제목에 해당 키워드가 포함된 항목만 표시 |
| 2 | 검색어 지우기 | 전체 목록 복원 |

**성공 기준:** 검색 결과가 입력 즉시 반영됨

---

### 시나리오 4 — 통계 대시보드로 진행 현황 파악 _(v5 신규)_

**행위자:** 박준호
**전제 조건:** Todo 항목 여러 개, 일부 완료 상태

| 단계 | 사용자 행동 | 시스템 반응 |
|------|------------|------------|
| 1 | 메인 페이지 접속 | 상단에 전체/완료/진행 중 개수 및 완료율 표시 |
| 2 | 항목 완료 처리 | 완료율 즉시 업데이트 |

**성공 기준:** 완료율이 할 일 완료/취소 시 실시간으로 변경됨

---

### 시나리오 5 — 기한 초과 항목 처리

**행위자:** 김지은
**전제 조건:** 완료일이 어제인 미완료 항목 존재

| 단계 | 사용자 행동 | 시스템 반응 |
|------|------------|------------|
| 1 | 메인 페이지 접속 | 기한 초과 항목 빨간 강조 표시 |
| 2 | 해당 항목 완료 처리 | 강조 해제, 취소선 처리 |
| 3 | 미완료 항목 완료일 수정 | 수정 저장 후 강조 해제 |

---

### 시나리오 6 — 인증 실패 및 재로그인

**행위자:** 박준호
**전제 조건:** JWT 토큰 만료

| 단계 | 사용자 행동 | 시스템 반응 |
|------|------------|------------|
| 1 | Todo 추가 시도 | 401 오류, 로그인 화면으로 이동 |
| 2 | 재로그인 | JWT 재발급, 메인 페이지 복귀 |

## 2. 예외 / 오류 시나리오

| 시나리오 | 트리거 | 시스템 처리 | 사용자 경험 |
|----------|--------|------------|------------|
| 중복 이메일 가입 | 기존 이메일로 가입 시도 | 400 반환 | "이미 사용 중인 이메일입니다" 오류 표시 |
| 잘못된 로그인 정보 | 틀린 비밀번호 입력 | 401 반환 | "이메일 또는 비밀번호가 올바르지 않습니다" |
| 제목 없이 Todo 추가 | 제목 빈 상태로 저장 | 422 반환 | "제목은 필수 항목입니다" |
| JWT 만료 | 토큰 유효기간 초과 | 401, 로그인 이동 | "세션이 만료되었습니다. 다시 로그인해주세요" |
| 검색 결과 없음 _(v5)_ | 매칭 항목 없는 키워드 | 빈 목록 반환 | "검색 결과가 없습니다" 안내 메시지 |
