## 1. 화면 흐름도

```
[미인증 사용자]
      │
      ├─── /register ──▶ 회원가입 페이지 ──▶ 가입 성공 ──▶ /login
      │
      └─── /login ──▶ 로그인 페이지
                            │
                       로그인 성공
                            │
                            ▼
[인증 사용자]  ──▶ / (메인 페이지 - 할 일 목록)
                            │
               ┌────────────┼────────────┐
               ▼            ▼            ▼
           할 일 추가    할 일 수정    할 일 삭제
           (폼 입력)    (모달 팝업)   (확인 다이얼로그)
```

**경로**: `GET /register`
**접근 조건**: 미인증 사용자

```
│          │  ┌───────────────────────────────┐  │               │
│          │  │ example@email.com             │  │               │
│          │  └───────────────────────────────┘  │               │
│          │                                     │               │
│          │  비밀번호                            │               │
```

### 상태별 UI

| 상태 | 표시 방법 |
|------|----------|

| 가입 성공 | 초록색 Toast 알림 → `/login` 리다이렉트 |

## 2.2 할 일 추가 폼

메인 페이지 상단에 항상 표시되는 인라인 폼입니다.

```
┌─────────────────────────────────────────────────────────────────┐
│  ➕ 새 할 일 추가                                                 │
│                                                                 │
│  제목 *                                                         │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ 할 일 제목을 입력하세요                                   │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
│  ┌──────────────────────┐  ┌──────────────────────┐            │
│  │ 우선순위              │  │ 계획 완료일           │            │
│  │ [보통 (medium)    ▼] │  │ [────-──-──  📅]     │            │
│  └──────────────────────┘  └──────────────────────┘            │
│                                                                 │
│  상세 내용                                                       │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │                                                         │   │
│  │ (선택 사항)                                              │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
│  ┌─────────────────┐                                           │
│  │   ➕ 할 일 추가  │  (Primary 버튼)                           │
│  └─────────────────┘                                           │
└─────────────────────────────────────────────────────────────────┘
```

**유효성 검사**:
- 제목: 필수 입력 (빈 값 제출 시 빨간 테두리 + "제목을 입력해주세요")
- 계획 완료일: 선택 사항, 오늘 이전 날짜 선택 불가 (datepicker min 설정)

## 2.4 기한 초과 상태 표시

계획 완료일(`due_date`)이 오늘 날짜를 지난 미완료 항목은 시각적으로 구분합니다.

```
┌── [빨간 배경 + 경고 아이콘] ────────────────────────────────────┐
│  ⚠️  ☐  제목 텍스트                          [✏️ 수정] [🗑️ 삭제] │
│      상세 내용                                                  │
│      ⚠️ 기한 초과: 2026-03-25 (3일 전)  |  🔴 높음  |  진행 중  │
└─────────────────────────────────────────────────────────────────┘
```

**기한 상태별 스타일**:

| 상태 | 배경색 | 테두리 | 날짜 텍스트 |
|------|--------|--------|-------------|
| 기한 초과 | `#fff5f5` (연한 빨강) | `#dc3545` | `text-danger` + ⚠️ |
| 오늘 마감 | `#fffbf0` (연한 주황) | `#fd7e14` | `text-warning` + 🔔 |
| 기한 여유 있음 | 흰색 | 우선순위 색상 | `text-muted` + 📅 |
| 기한 없음 | 흰색 | 우선순위 색상 | 표시 없음 |

## 공통 컴포넌트

### 네비게이션 바

```
┌─────────────────────────────────────────────────────────────────┐
│  📋 My Todo List           [사용자명 ▼]  [🔔 알림]  [로그아웃]   │
└─────────────────────────────────────────────────────────────────┘
```

- 미인증 상태: `[로그인]` `[회원가입]` 버튼 표시
- 인증 상태: 사용자명 드롭다운 + 로그아웃 버튼 표시

### Toast 알림 (우측 하단)

```
                              ┌────────────────────────┐
                              │ ✅ 할 일이 추가되었습니다! │
                              └────────────────────────┘
```

- 성공: 초록색 (`bg-success`)
- 오류: 빨간색 (`bg-danger`)
- 3초 후 자동 사라짐

### 삭제 확인 다이얼로그

```
       ┌──────────────────────────────────────────┐
       │  🗑️ 할 일 삭제                          ✕ │
       ├──────────────────────────────────────────┤
       │  "[제목]"을(를) 삭제하시겠습니까?          │
```

- 우선순위 + 날짜 선택: 한 줄 2열 그리드
- 통계: 간소화된 1줄 표시
- 필터 탭: 아이콘 + 텍스트 축약

## API ↔ 화면 매핑

| 화면 동작 | API 호출 | 성공 처리 |
|----------|---------|----------|
| 회원가입 폼 제출 | `POST /auth/register` | `/login` 리다이렉트 |
| 로그인 폼 제출 | `POST /auth/login` | `/` 리다이렉트 + JWT 저장 |
| 로그아웃 버튼 | (클라이언트) JWT 삭제 | `/login` 리다이렉트 |
| 페이지 로드 | `GET /todos` | 목록 렌더링 |
| 할 일 추가 폼 제출 | `POST /todos` | 목록 최상단에 추가 |
| 체크박스 클릭 | `PATCH /todos/{id}/toggle` | 카드 스타일 전환 |
| 수정 저장 | `PUT /todos/{id}` | 카드 내용 갱신 |
| 삭제 확인 | `DELETE /todos/{id}` | 카드 제거 |
